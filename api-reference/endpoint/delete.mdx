### 📌 Endpoint

- **Method:** `DELETE`
- **URL:** `/api/api-keys/{id}`
- **Auth Required:** Yes (Bearer Token)
- **Permissions:** User must own the API key or be an admin.

---

### 🔐 Authentication

> Requires Bearer Token in the `Authorization` header:
> "
>
> ---
>
> ### 🔸 Parameters
>
> | Name | Type | Location | Description            |
>
> |------|------|----------|------------------------|
>
> | id   | UUID | Path     | ID of the API key to be deleted |
>
> ---
>
> ### ✅ Success Response
>
> **HTTP 204 No Content**
>
> No body is returned on success.
>
> ---
>
> ### ❌ Error Responses
>
> - **401 Unauthorized** – Token is missing or invalid.
> - **403 Forbidden** – The API key does not belong to you.
> - **404 Not Found** – The specified `id` was not found.
>
> ---
>
