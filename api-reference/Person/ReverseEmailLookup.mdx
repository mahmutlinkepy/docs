---
title: "Reverse Email Lookup"
description: "This operation allows you to find data from an email. It returns data profile and company information as the response. This operation consume 1 credit. "
---

## 🔐 Authorizations

| Name   | Type   | In    | Required |
| :----- | :----- | :---- | :------- |
| apikey | string | query | ✅        |

`apikey` parametresi, geliştirici panelinden aldığınız API anahtarıdır ve zorunludur.

## 📥 Query Parameters

| Name            | Type   | Required | Description                         |
| :-------------- | :----- | :------- | :---------------------------------- |
| `email`         | string | ✅        | Sorgulanacak e-posta adresi         |
| `firstName`     | string | ❌        | (Opsiyonel) Kişinin adı             |
| `lastName`      | string | ❌        | (Opsiyonel) Kişinin soyadı          |
| `companyDomain` | string | ❌        | (Opsiyonel) Şirket domaini / URL'si |
| `companyName`   | string | ❌        | (Opsiyonel) Şirket adı              |

---

## ✅ Response — 200 OK

```
json
```

`{   "success": true,   "email": "johndoe@example.com",   "emailType": "personal",   "person": {     "fullName": "John Doe",     "location": "New York, USA",     "title": "Software Engineer",     "profilePicture": "https://example.com/johndoe.jpg",     "linkedInUrl": "https://linkedin.com/in/johndoe"   },   "credits_left": 1987,   "rate_limit_left": 94 } `

---

## 📤 Response Fields

| Field             | Type    | Description                                                     |
| :---------------- | :------ | :-------------------------------------------------------------- |
| `success`         | boolean | API isteğinin başarılı olup olmadığını belirtir                 |
| `email`           | string  | Sorgulanan e-posta adresi                                       |
| `emailType`       | string  | E-posta türü (`personal`, `work`, vb.)                          |
| `person`          | object  | Kişiye ait profil bilgilerini içeren nesne                      |
| `credits_left`    | integer | Bu sorgudan sonra kalan kullanılabilir kredi sayısı             |
| `rate_limit_left` | integer | Bu sorgudan sonra günlük kullanım limitinden kalan istek sayısı |

---

## 💻 Example Request

```
bash
```

`curl --request GET \   --url 'https://api.example.com/enrichment?email=johndoe@example.com&apikey=YOUR_API_KEY' `
